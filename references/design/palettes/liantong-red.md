# 配色：联通红（liantong-red）

> 来源：联通新一代知识管理平台（原 skill 默认配色）。

| Token | 色值 | 自然语言描述词 | 用途 |
|:---|:---|:---|:---|
| `{PRIMARY}` | `#E60012` | 中国红 / vivid red | 标题重点、核心高亮、数据流 |
| `{SECONDARY}` | `#4A4A4A` | 深灰 | 底层结构、底座 |
| `{ACCENT}` | `#FF6600` | 活力橙 | 少量强调 |
| `{BG}` | `#F5F7FA` → `#FFFFFF` | 浅灰白渐变 | 背景 |
| `{TITLE}` | `#333333` | 深灰 | 标题 |
| `{BODY}` | `#666666` | 中灰 | 正文 |
| `{OK}` | `#2E9E6B` | 绿 | 正确/完成 |
| `{WARN}` | `#FF6600` | 橙 | 警示 |

## 提示词配色描述段

```
Color scheme: light gray-white gradient background, vivid red (#E60012) as the only
accent color for highlights and data flows, dark gray structural elements,
neutral gray text. No other hues, no colorful icons.
```
