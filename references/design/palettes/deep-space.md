# 配色：深空青（deep-space）⚠️深色配色

> 来源：宁波外理智能理货位置复核系统。唯一深色配色，需用户明确接受深色背景。

| Token | 色值 | 自然语言描述词 | 用途 |
|:---|:---|:---|:---|
| `{PRIMARY}` | `#35E0C8` | 荧光青 / glowing cyan-teal | 发光线框、数据流、成功态 |
| `{SECONDARY}` | `#4FC3F7` | 冰蓝 | 次级线框、图标 |
| `{ACCENT}` | `#FFB300` | 琥珀 | 警示、关键风险高亮 |
| `{BG}` | `#0A1C2C` → `#0B1E2D` | 深藏青渐变 | 背景 |
| `{TITLE}` | `#E6F7FF` | 近白浅青 | 标题 |
| `{BODY}` | `#7FA8C9` | 灰青 | 次要文字 |
| `{OK}` | `#35E0C8` | 荧光青 | 正确/完成 |
| `{WARN}` | `#FFB300` | 琥珀 | 警示 |

## 提示词配色描述段

```
Color scheme: deep dark navy background (#0A1C2C), glowing cyan-teal (#35E0C8) wireframes
and highlights, ice blue secondary lines, amber (#FFB300) warnings only, near-white text
with high contrast. Dark sci-fi palette.
```
