# 配色：科技蓝（tech-blue）

> 来源：兴业银行审计智能体 / 明东码头方案。

| Token | 色值 | 自然语言描述词 | 用途 |
|:---|:---|:---|:---|
| `{PRIMARY}` | `#2B7BFF` | 科技蓝 / vivid tech blue | 重点、核心图标、数据流 |
| `{SECONDARY}` | `#63A4E2` | 中蓝 | 渐变卡片、次级图标 |
| `{ACCENT}` | `#25C4B0` | 青绿 | 成功态、小点缀 |
| `{BG}` | `#E7F0FA` → `#FFFFFF` | 浅蓝白渐变 | 背景 |
| `{TITLE}` | `#1A2B4C` | 深蓝黑 | 标题 |
| `{BODY}` | `#5A6B85` | 蓝灰 | 正文 |
| `{OK}` | `#25C4B0` | 青绿 | 正确/完成 |
| `{WARN}` | `#FFB300` | 琥珀 | 警示 |

## 提示词配色描述段

```
Color scheme: light blue-white gradient background, vivid tech blue (#2B7BFF) primary
accent with mid-blue gradients, dark navy titles, blue-gray body text. Single blue
family, no colorful icons.
```
