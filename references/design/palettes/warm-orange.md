# 配色：暖橙（warm-orange）

> 来源：外一知识中台介绍。

| Token | 色值 | 自然语言描述词 | 用途 |
|:---|:---|:---|:---|
| `{PRIMARY}` | `#EC6309` | 橙 / warm orange | 重点词、核心高亮、箭头 |
| `{SECONDARY}` | `#D9CDB8` | 米色 | 线描装饰、几何纹样 |
| `{ACCENT}` | `#F5A623` | 金橙 | 渐变高光、发光核心 |
| `{BG}` | `#F2EFE7` / `#F7F7F7` | 米白暖调 / 浅灰白 | 封面 / 内容页背景 |
| `{TITLE}` | `#2E2E2E` | 近黑 | 标题 |
| `{BODY}` | `#666666` | 中灰 | 正文 |
| `{OK}` | `#5E9E4A` | 绿 | 正确/完成 |
| `{WARN}` | `#D8451A` | 深橙红 | 警示 |

## 提示词配色描述段

```
Color scheme: warm off-white background, warm orange (#EC6309) as the only vivid accent,
beige line decorations, near-black titles, gray body text. Warm, friendly, restrained.
```
